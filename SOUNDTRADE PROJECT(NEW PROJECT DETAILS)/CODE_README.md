# SoundTrade Installation/Requirements Guide

### Requirements:

- MongoDB Community Server
- Mongosh

### Instructions:

1. Download all files in the "SOUNDTRADE CODE" folder and place them in a directory you will remember.
2. Open Command Prompt and navigate to the directory with the program files.
3. Use the command "node app.js" to run the program.
4. in any browser, type "localhost:3000" in the search bar.
