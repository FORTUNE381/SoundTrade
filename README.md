<h1 align="center">Trade My Skills (SoundTrade)</h1>
<p align="center">A Community-Based Skill Sharing Platform</p>


<h2>Overview</h2>
SoundTrade is a web-based platform where users can both learn and teach skills.
Instead of traditional one-way learning, this platform promotes peer-to-peer exchange, where everyone can contribute knowledge regardless of expertise level.

## Problem Statement(WHY)
Many learners truggle with:
- Lack of guidance <br>
- Expensive learning resources <br>
- Learning alone without support <br>

At the same time, many individuals have valuable skills but lack confidence or a platform to share them.

## Solution
SoundTrade provides a space where users can search for music & instrumental skills they want to learn, share skills they already have, and connect with others for mutual learning. This encourages collaboration, confidence-building, and community-based learning.

## Features
- User signup and login
- Search for skills to learn
- Post skills to share
- View matches or available skill posts
- User dashboard
- Messaging or trade coordination between users

## MVP(Minimum Viable Product)
Our project includes at least one complete MVP, featuring:

- User Authentication (Login/Signup)
- Skill Search (know vs. want to learn)
- Post Skills (if no match found)
- Basic interaction between users
- Dashboard to view activity

 This MVP demonstrates the core functionality of the platform and validates our main idea.

## Installation & Setup

1. Clone the repository:
```bash
git clone https://github.com/FORTUNE381/SoundTrade.git
```
2. Navigate to the project folder:
```bash
cd "SOUNDTRADE CODE"
```
3. Install dependencies:  
```bash
node app.js
```
4. Open in your browser:
```bash
http://localhost:3000
```
> Make sure MongoDB is running locally before starting the server.
